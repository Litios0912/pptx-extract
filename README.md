Herramienta para extraer y analizar contenido de archivos PowerPoint (.pptx).

## ✨ Características

- 📖 Extrae texto de todas las diapositivas
- 🖼️ Exporta imágenes y medios
- 📊 Analiza estructura de la presentación
- 📝 Genera resumen en Markdown

## 🚀 Instalación

```bash
pip install -r requirements.txt
```
